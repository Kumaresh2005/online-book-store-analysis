 Online Book Store Analysis (SQL Project)

This project focuses on analyzing an online bookstore database using SQL to generate meaningful business insights. The database includes tables such as customers, books, orders, and order details.

In this project, I performed data cleaning and analysis using SQL concepts like SELECT statements, WHERE conditions, GROUP BY, ORDER BY, JOINS (INNER, LEFT, RIGHT), subqueries, aggregate functions (SUM, AVG, COUNT), and window functions.

🔍 Key Analysis Performed:

Identified top-selling books and most profitable categories

Found most frequent and high-value customers

Analyzed monthly and yearly sales trends

Calculated total revenue and average order value

Determined most frequently ordered books

Explored customer region-wise performance

🛠 Tools & Technologies:

SQL (MySQL / PostgreSQL)

SQL Workbench
